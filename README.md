# Campo Minado — Android

Jogo simples de campo minado desenvolvido em B4A (Basic4Android).

## Conteúdo do repositório

Arquivos:
- Campo minado.apk (instalador para Android)
- print (no emulador)
- Arquivos do projeto B4A

## Como jogar

O jogo funciona tocando os botões espalhados pelo campo.

**Toque duplo rápido:** revela o que há na célula. Se for uma bomba, fim de jogo.

**Toque único:** marca a célula com uma bandeira. Toque novamente na bandeira para desmarcar.

Você vence ao marcar corretamente as 10 bombas. 

Toque em **Iniciar** para começar.
